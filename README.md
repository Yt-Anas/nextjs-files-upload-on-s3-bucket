# 📁 Next.js File Upload to AWS S3

This project allows users to upload files through a Next.js application.  
Uploaded files are stored in AWS S3 and a public URL is generated for access.

## 🚀 Features

- Upload files from browser
- Store files in AWS S3
- Generate public URL
- Simple UI

## 🛠 Tech Stack

- Next.js
- AWS S3
- AWS SDK

## 📦 How It Works

1. User selects file  
2. File is sent to API  
3. Uploaded to S3  
4. URL returned  

## 📌 Future Improvements

- Pre-signed URLs  
- File validation  
- Upload progress
